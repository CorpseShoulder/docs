Minimum Requirements - 2 Nodes

Server (Node) Specifications
CPU
  • Intel or AMD 64 bit processor
  • Consistent processor family per cluster
  • Note: A cluster is a minimum of 2 servers (nodes) with like architecture, multiple clusters can live within a system
  • Virtualization support (VT-x, AMD-V/AMD-SVM)
Memory
  • Minimum of 8 GB for VergeIO operations (OS and Appserver)
  • Designated memory for guests
  • 1 GB of RAM per 1 TB of storage in each server (node) participating in storage
Storage
  • Host Bus Adapter (HBA) or RAID controller that supports JBOD mode (NO RAID) and certified to work with the latest LTS Linux Kernel
  • 1 High Endurance (~3 DWPD) NVME of 240 GB or greater to be used as dedicated storage for environment metadata
  • Note: This is only required by the first two servers (nodes) which act as the controllers for the environment
  • 1 NVME or SATA/SAS SSD of the same size per server (node) for guest usage (more are recommended for performance purposes)
  • Note: HDD‚Äôs can be used in place of NVME or SATA/SAS SSD‚Äôs with an acknowledged performance impact
