The Verge.io platform records user activity and displays it in the System Log.  All logs in the system can be reviewed from the Main Dashboard and then click the Logs button in the left navigation menu.  From this view, all logs in the system can be reviewed.

As the Verge.io platform will only keep logged activity for a maximum of 45 days, logs can be exported out of the system by configuring a 3rd party log collection, for long-term record keeping.  Setting up a 3rd party system logger can be configured from the Main Dashboard and then click on System in the left navigation menu.  Inside the System menu, click on Settings in the left navigation menu.  Once inside the Settings options, find the setting labeled 'Remote syslog server (tcp: @@name/ip:port, udp: @name/ip:port)'.  Configure that setting following the implied syntax in the setting name.

Additionally, in many areas of the platform, such as a specific VM dashboard, there is a Logs button to view only logs specific to that context.  This is often helpful in filtering out many logged events that aren't relevant to a specific area of the platform.
