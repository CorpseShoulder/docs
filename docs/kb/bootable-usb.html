How to burn the Verge installer onto a USB drive

Windows:

  1. Download the latest version of the Verge.io installation ISO.
    1. https://updates.verge.io/public/verge-install-4.9.2.iso
    2. Disclaimer: This may not be the most recent version. Please talk to your account representative to make sure you have the latest version.
  2. Insert a USB disk into the computer. This USB drive will be overwritten.
  3. Launch Rufus.
    1. https://rufus.ie/en/
  1. Once Rufus is loaded, select the USB device to write to for the area labeled Device.
  2. Click the CD-ROM icon next to ‘Create a bootable disk using’ and then select the Verge.io installation ISO image.
  3. Once the ISO image is selected, change the setting for ’Create a bootable disk image using’ to DD Image.
  4. After selecting DD Image mode, click Start and Rufus will begin building the bootable USB device.

 

 

Apple MacOS:

  1. Download the latest version of the Verge.io installation ISO.
    1. https://updates.verge.io/public/verge-install-4.9.2.iso
    2. Disclaimer: This may not be the most recent version. Please talk to your account representative to make sure you have the latest version.
  1. Download and install the BalenaEtcher image flasher for the macOS.
    1. https://www.balena.io/etcher/
  2. Startup the BalenaEtcher app.
  3. Click on Flash from file.
  1. Pick the VergeOS ISO file that you downloaded.
  2. Select your target USB disk, but NOT the hidden drives
  3. Click on Flash!
  1. This will build your bootable flash drive on your USB stick using the VergeOS ISO installer. This process usually takes a few minutes to complete and then you can “eject” your USB flash drive to boot and start the installation on your servers.

 

Linux Mint:

  1. Download the latest version of the Verge.io installation ISO
    1. https://updates.verge.io/public/verge-install-4.9.2.iso
    2. Disclaimer: This may not be the most recent version. Please talk to your account representative to make sure you have the latest version.
  2. Using Mint’s file browser, navigate to the location of the downloaded ISO
  3. Right-click on the ISO and choose ‘Make bootable USB stick’
  4. As USB Image Writer loads, select the USB media to write the installation file to
  5. As a security measure, Linux will prompt you to supply an admin password
  6. Click Authenticate to begin writing the installation to USB
  7. Wait while the installation is written to USB.
  8. When finished this message will appear.
